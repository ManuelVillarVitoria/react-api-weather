# `Obtaining the Weather since an API`

_The project consists of doing the following:_

_1.-Creating the 'Header'._

_2.-Creating,reading and validating the form._

_3.-Making a request to the API throught  a function and obtaining the results of it._

_4.-Displaying the results of the API in the DOM and verifying there are results._

_5.-Documenting the App._

_- Have been used **Functional Components**, **Hooks useState & useEffect**, **PropTypes** and  **Fetch API + Async/Await** in this project._
